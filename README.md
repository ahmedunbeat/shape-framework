# BACKEND: Geometric Shape Framework (PHP OOP)

## The OOP design was done with:
    - interface CalculatedInterface use in creating the contract for all the 
      shapes framework which must include (Area and Perimeter)
      
    - Access modifier (public) to make the property visible to all classes
    
    - abstract class use as baseline to create part of the code($name) for all shapes 
      to be continue with other functions
      
    - PHP 8 use in constucting the methods
    
    - Inheritance of parent class using [extend] keyword for individual class
      of shapes - Polymorphism
      
    - foreach shape(circle, rectangle, triangle) an instance was created in the 
      properties for area and perimeter calculation.
